## 报告安全漏洞 / Reporting a Vulnerability

我们非常重视安全问题。如果您发现任何安全漏洞或潜在的安全问题，请通过以下方式报告：

We take security seriously. If you discover any security vulnerability or potential security concern, please report it through the following channel:

**通过 GitHub Issues 提交 / Submit via GitHub Issues:**
https://github.com/qiuhaomem/HawkEye-Mem/issues

### 我们的承诺 / Our Commitment

- **48 小时内确认**：我们承诺在收到报告后的 **48 小时内**确认收到您的报告。
  **Confirmation within 48 hours**: We commit to acknowledging receipt of your report within **48 hours**.

- **及时响应**：我们将尽快评估问题并制定修复计划。
  **Timely response**: We will assess the issue and develop a remediation plan as soon as possible.

- **公开致谢**：我们感谢所有负责任披露安全问题的研究人员，并将在修复后公开致谢（经您同意）。
  **Public acknowledgment**: We thank all security researchers who responsibly disclose issues, and will publicly acknowledge you upon fix (with your consent).

### 注意事项 / Notes

- 本项目目前规模较小，**暂不支持 CVE 编号分配**。
  This project is currently small in scale and **does not support CVE assignment at this time**.

- 请勿在公开渠道（如 Issues 评论区、Pull Requests、讨论区）披露未修复的安全漏洞。
  Please do not disclose unfixed security vulnerabilities in public channels (such as Issue comments, Pull Requests, or discussion forums).

- 我们鼓励所有安全研究者和社区成员积极参与本项目的安全建设。
  We encourage all security researchers and community members to actively participate in the security development of this project.

感谢您帮助保护 HawkEye-Mem 及其社区的安全！

Thank you for helping keep HawkEye-Mem and its community safe!
