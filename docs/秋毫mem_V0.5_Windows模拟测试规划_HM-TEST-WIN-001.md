## 一、测试目标

| 目标 | 说明 |
|------|------|
| 兼容性 | Token 审计在 Win7/8/10/11 下的行为一致性 |
| 资源限制 | 不同硬件配置下的性能表现 |
| 路径差异 | Windows `\` vs Linux `/` 路径处理 |
| 文件系统 | NTFS vs ext4 权限差异 |
| Python 版本 | Win7 最高支持 Python 3.8，Win11 支持 3.12+ |

## 二、环境模拟方案

### 2.1 方案对比

| 方案 | 准确度 | 复杂度 | 是否可用 |
|------|--------|--------|----------|
| Docker + Wine | ⭐⭐⭐ | 中 | ❌ 无 Docker |
| QEMU 虚拟机 | ⭐⭐⭐⭐⭐ | 高 | ❌ 无 QEMU |
| VirtualBox | ⭐⭐⭐⭐ | 高 | ❌ 无 VBox |
| **纯 Python 模拟** | ⭐⭐ | **低** | **✅ 可用** |

**结论**: 当前环境只能用纯 Python 模拟。如需更准确测试，需在真实 Windows 机器上运行。

## 三、测试矩阵

### 3.1 资源限制测试

| Windows 版本 | CPU | 内存 | 测试内容 |
|-------------|-----|------|----------|
| Win7 | 2核 | 2GB | 基础功能 + 并发 |
| Win8 | 4核 | 4GB | 基础功能 + 并发 |
| Win10 | 4核 | 8GB | 基础功能 + 并发 + 大数据 |
| Win11 | 8核 | 16GB | 全功能 + 极限并发 |

### 3.2 路径兼容性测试

| 测试项 | Win7 | Win8 | Win10 | Win11 |
|--------|------|------|-------|-------|
| 路径分隔符 `\` | ✅ | ✅ | ✅ | ✅ |
| 长路径支持 (>260字符) | ❌ | ❌ | ✅ | ✅ |
| UNC 路径 `\\server\share` | ✅ | ✅ | ✅ | ✅ |
| 中文路径 | ⚠️ | ⚠️ | ✅ | ✅ |

### 3.3 Python 版本兼容性

| Windows | 最高 Python | 状态 |
|---------|-------------|------|
| Win7 | 3.8 | ⚠️ 需测试 3.8 |
| Win8 | 3.10 | ✅ 需测试 3.10 |
| Win10 | 3.11 | ✅ 当前版本 |
| Win11 | 3.12+ | ✅ 需测试 3.12 |
