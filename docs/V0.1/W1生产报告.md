## 四、遇到的问题与解决方案

| 问题 | 影响 | 解决方案 |
|:----|:-----|:---------|
| 系统缺少 libc6-dev 开发包 | 无法编译任何 Rust/C 代码 | apt download + sysroot 解耦：从 deb 包提取 crt/库文件到用户目录，通过 RUSTFLAGS 指定 --sysroot |
| gcc-15 链接器硬编码系统库路径 | 链接失败找不到 libc_nonshared.a | 创建自定义 sysroot 目录，配置 gcc wrapper 脚本 |
| 工具链验证环境无 sudo 权限 | 无法直接 apt install | 纯用户空间搭建开发环境，所有依赖在用户目录内解决 |
